┌──────────────────────────────────────────────────────────────────────────────┐
│  🚀 QUICK START COMMANDS                                                     │
└──────────────────────────────────────────────────────────────────────────────┘

    1. Install Dependencies
       $ pip install -r requirements.txt

    2. Verify Setup
       $ python check_training_setup.py

    3. Train Model (Choose one method)
       
       Method A: Direct Python
       $ python -m model_training.train_regression_model
       
       Method B: Via API
       $ python main.py  # Start backend
       $ curl -X POST http://localhost:8000/api/train-model
       
    4. Check Model Status
       $ curl http://localhost:8000/api/model-info

    5. Test Evaluation (Uses ML model automatically!)
       $ curl -X POST http://localhost:8000/api/evaluate \
         -F "question=Your question" \
         -F "reference_answer=Reference answer" \
         -F "answer_image=@student_answer.jpg"
